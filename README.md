## Installation
Use `pip` to install this tool

    pip install colorcalendar


## Usage

See help:

    colorcalendar -h

Basic:

    colorcalendar --date 2018-2-20

or  

    colorcalendar -y 2018 -m 2 -t 20

Change color:

    colorcalendar -f y -b b

Color list:
[Refer to](https://github.com/houluy/colorline)

Without color:  

    colorcalendar -c
